# logbook

logbook is command line tool for keeping diary, made in golang.

## Install

```sh
$ git clone https://github.com/yuuu/logbook
$ cd logbook
$ go get
$ go build
$ go install
```

## Usage

### Keep a diary

```sh
$ logbook keep
(An editor(default:vi) will launch.)
(You can input text of entry.)
```
